## схемы тегирования

### Схема тегирования `--tag-ci`

Тег формируется из переменных среды:

* `CI_BUILD_ID`;
* `TRAVIS_BUILD_NUMBER`.

### Схема тегирования `--tag-build-id`

Тег формируется из переменных среды:

* `CI_BUILD_REF_NAME`, `CI_BUILD_TAG`;
* `TRAVIS_BRANCH`, `TRAVIS_TAG`.
